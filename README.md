# 🏠 Smart Property Analytics & Visualization

An interactive real estate analytics dashboard built with **Streamlit, Plotly, and Python** to explore housing market trends, pricing patterns, and property insights through intuitive visualizations.

🔗 **Live Demo:**
https://smart-property-analytics-visualization.streamlit.app/

---

## 🚀 Overview

This project provides a data-driven interface to analyze real estate listings and uncover meaningful insights such as price distribution, sector-wise trends, furnishing impact, and demand patterns.

It is designed to simulate how modern analytics tools help users make informed decisions in the property market.

---

## ✨ Key Features

* 📊 **Market Snapshot**

  * Quick overview of listings, median price, area, and price per sqft

* 🗺️ **Geo-based Visualization**

  * Sector-wise pricing insights using interactive maps

* ☁️ **Feature Wordcloud**

  * Highlights commonly occurring amenities and property features

* 📈 **Area vs Price Analysis**

  * Understand how property size impacts pricing across categories

* 🏢 **BHK Demand Insights**

  * Distribution and comparison of different property configurations

* 💎 **Luxury Score Analysis**

  * Visual correlation between luxury levels and pricing

* 🪑 **Furnishing Impact**

  * Comparison of furnished vs unfurnished property pricing

* 🔥 **Heatmaps & Trends**

  * Sector vs BHK pricing patterns
  * Price distribution across property types

---

## 🛠️ Tech Stack

* **Frontend/UI:** Streamlit
* **Visualization:** Plotly, Matplotlib, Seaborn
* **Data Processing:** Pandas, NumPy
* **Additional:** WordCloud

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Ayushyash24/Smart-Property-Analytics-Visualization-.git
cd Smart-Property-Analytics-Visualization-
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run Home.py
```
